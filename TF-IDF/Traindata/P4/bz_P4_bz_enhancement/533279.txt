If we have a Set of resources, we should have a helper method to create a Set of Links to those resources.

Example:

HashSet<Person> contributors = new HashSet<Person>();
...

final HashSet<Link> contributorLinks = new HashSet<>();
for (Person contributorItem : contributors) {
	contributorLinks.add(new Link(contributorItem.getAbout()));
}

changeRequest.setContributor(contributorLinks);
