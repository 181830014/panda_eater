Pressing on the 'reply' button on a comment in the task editor wraps comment text.  Short strings that don't normally wrap in the task editor are wrapped, making the quoted text hard to read.

For example, the following comment:

bq. 
David, would you be interested in adding an ant target for running the standalone tests?

is wrapped as follows:

bq. 
(In reply to comment #3)
> David, would you be interested in adding an ant target for running the
> standalone tests?
