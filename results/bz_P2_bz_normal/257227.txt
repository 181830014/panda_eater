Build ID: 3.4.1

Steps To Reproduce:
1. Open the project's Build Path properties page.
2. Ensure the project's folder is selected as the Build Path root.
3. Click Link Source.
4. Select a folder and click Finish.
This will result in a nesting conflict.


More information:
In the Link Source dialog, the 'Update exclusion filters' option should be selected by default so that the nesting conflict will not occur.
